Movement Blocks (Drive)

drive forward

Starts moving the robot forward continuously without a distance limit.

drive forward distance

Moves the robot forward for a specific distance (in cm) and then stops.

drive distance at speed

Moves for a defined distance while setting the motor power as a percentage.

drive until at speed

Moves at a defined speed until an external condition (sensor) is met.

turn degree at speed

Turns the robot to a precise angle at the selected speed.

turn until at speed

Turns at a set speed until a condition is met (e.g., until a line is detected).

set motor power

Sets separate power levels for each motor (Left/Right) for full manual control.

set heading to degree

Resets or sets the internal compass heading to a specific degree value.

set default speed

Sets the constant working speed for all subsequent movement actions.

stop moving

Immediately brakes and stops all robot movements.

Lighting (LEDs)

set led color

Sets the color of the robot's LED light based on your selection.

led detected color

Changes the LED color to match the color currently detected by the sensor.

turn off led

Turns off the robot's LED light.

Control Blocks

forever

A loop that runs the code inside it over and over again indefinitely.

if then

Executes an action only if a specific condition is met.

if then else

Executes action A if the condition is met, and action B if it is not.

repeat until

Repeats a block of code until a specific stopping condition is met.

wait (seconds)

Pauses the program for a set number of seconds before moving to the next block.

wait until

Pauses the program execution at this point until a specific condition becomes true.

stop program

Immediately ends the execution of the entire program.

Sensors

distance from obstacle

Returns the exact distance to the nearest obstacle in centimeters.

gyro angle

Displays the current turning angle of the robot relative to its starting point.

touch sensor pressed?

Checks if the touch sensor is currently being pressed (returns true or false).

touching color?

Checks if the sensor is over a specific color selected on the surface.

wheel circumference

Technical data used for precise movement calculations based on wheel size.

Logic and Math

logic and / or

Allows combining multiple conditions (e.g., move if there is a black line AND no obstacle).

comparison

Compares values: greater than, less than, or equal to.

math operators

Performs arithmetic operations like addition, subtraction, multiplication, and division.

logic not

Inverts the result of a condition (if it was true, it becomes false).

integer of

Rounds a decimal number to the nearest whole integer.

true / false

Fixed logical values representing "True" or "False".
